# Árbol Binario - Práctico Experimental

## Descripción
Este proyecto implementa un árbol binario en Python con un menú interactivo en consola.

## Funciones del sistema
- Insertar nodos en el árbol
- Mostrar árbol en recorrido preorden
- Menú interactivo para el usuario

## Lenguaje utilizado
- Python 3

## Ejecución
Ejecutar el programa con el siguiente comando:

python arbol.py
